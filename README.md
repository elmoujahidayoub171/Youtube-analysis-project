# YouTube Statistics Analysis and Visualization

## Project Overview

This project focuses on the analysis and visualization of YouTube statistics data using **Python** and **Jupyter notebooks**. It is designed to explore and understand patterns, trends, and relationships within YouTube-related metrics through both **bivariate** and **multivariate** analysis.


## Objectives

- Analyze YouTube statistics data in a structured way
- Perform **univariate analysis** to study the distribution of variables
- Perform **bivariate analysis** to study relationships between two variables
- Perform **multivariate analysis** to explore interactions among multiple variables
- Visualize trends and patterns for better interpretation of the data
- Organize the workflow from raw data to processed analytical outputs
